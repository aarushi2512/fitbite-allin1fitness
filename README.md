# Fitbite: An Intelligent Wellness & Nutrition Ecosystem

**Fitbite** represents a paradigm shift in health applications by seamlessly integrating fitness tracking and precision nutrition into a single, user-centric platform. Designed to solve the challenge of maintaining a healthy lifestyle amidst modern, fast-paced schedules, the system caters to both seasoned enthusiasts and wellness beginners.

---

## 🚀 Key Features & Logic

### 1. Intelligent Recommendation Engine
* **Inventory-Aware Suggestions:** Uses advanced algorithms to suggest recipes based specifically on available items in the user's kitchen.
* **Goal-Oriented Nutrition:** Crafts tailored meal plans based on individual fitness goals, specifications, and dietary restrictions.
* **Weight Management:** Provides meticulously curated recipes specifically for gaining, losing, or maintaining weight.

### 2. Health Analytics & Biometrics
* **Metabolic Tracking:** Implements automated diet planning based on calculated **Basal Metabolic Rate (BMR)** and **Body Mass Index (BMI)** values.
* **Nutritional Transparency:** Every recipe is accompanied by detailed nutritional data, ingredient lists, and step-by-step preparation guides.

### 3. Geolocation Services
* **Nutritionist Locator:** Leverages real-time location detection to find and navigate to the nearest certified nutritionists within a **10km range**.
* **Dynamic Updating:** The system continuously updates location data at regular intervals to ensure accuracy.

## 🛠️ Technical Architecture

### Core Stack
* **Language:** Java
* **Development Environment:** Android Studio
* **Backend/Database:** Firebase Real Time Database
* **Cloud Infrastructure:** Firebase Cloud Hosting

### Module Split
The project is architected into two distinct modules for streamlined management:
1. **`Allin1FitnessUser`**: The client-side application for end-user interaction and fitness tracking.
2. **`Allin1fitnessAdmin`**: The administrative dashboard for managing the global recipe repository and user specifications.
